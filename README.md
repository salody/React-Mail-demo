# React 示例项目

  This is a React Inbox demo based on [React Demo]
![screenshot](./screenshot.png)

### Installation

You need cross-env installed globally:

```sh
$ npm i cross-env -g
```

```sh
$ git clone https://github.com/salody/React-Mail-demo.git
$ npm install
$ npm start
```
